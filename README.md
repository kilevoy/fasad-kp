# 🧱 Генератор КП по фасадной системе ИНСИ

> Результат расчёта фасада → готовое коммерческое предложение за секунды: спецификация из геометрии, смета по реальному прайсу ИНСИ, декомпозиция цены, ₽/м², условия и PDF.

```
📐 Геометрия фасада → 🧱 Спецификация (BoM) → 🧮 Смета (прайс, скидка, НДС) → 📄 КП (веб + PDF)
```

Менеджер по фасадам тратит часы, чтобы превратить расчёт в коммерческое предложение в Word/Excel. Этот инструмент берёт результат расчёта фасадной системы и собирает профессиональное КП мгновенно: спецификация считается **из реальной геометрии фасада**, а не «на глаз», цены — из актуального прайса ИНСИ.

---

## Чем отличается от обычного генератора КП

- **Инженерная спецификация из геометрии** — кассеты (м²/шт), подсистема КВГУ + НПГ (шт/п.м), крепёж, мембрана (нахлёст), утеплитель (м³) — вычисляются из габаритов фасадов и проёмов.
- **Реальный прайс ИНСИ** (март 2026) — без выдуманных цен.
- **Декомпозиция стоимости** по разделам сметы — прозрачность для закупщика.
- **₽/м²** — ключевая метрика для фасадчика.
- **Границы поставки** — что входит / не входит / риски и допущения.
- **Коммерческая модель в реальном времени** — запас, скидка, монтаж, НДС, график оплаты.

## Что делает

- **Геометрия** (`bom.calculate_geometry`) — площади фасадов и проёмов, чистая площадь облицовки.
- **Спецификация** (`bom.build_bom`) — натуральные объёмы материалов по правилам ИНСИ.
- **Смета** (`compose`) — суммы, скидка, НДС, ₽/м², график оплаты. Покрыто тестами.
- **Рендер** (`render`) — самодостаточная HTML-страница КП, готовая к печати в PDF.

## Установка и запуск (ядро)

```bash
pip install -r requirements.txt

# собрать КП из результата расчёта фасада (выгрузка калькулятора ИНСИ):
python -m kp sample/calc-result.json --city Екатеринбург --object "Административный корпус" \
  --client "ООО Альфа-Строй" --number КП-007 --out out/kp.html
```

Результат — `out/kp.html`: откройте в браузере → «Печать» → сохранить в PDF.

## Веб-демо

```bash
cd web
npm install
npm run dev      # http://localhost:5173
npm run build    # сборка в web/dist
```

Интерактивное КП: слайдеры коммерческих допущений, живой пересчёт сметы, декомпозиция цены, границы поставки, график оплаты, светлая/тёмная тема.

## Архитектура

| Модуль | Роль |
|--------|------|
| `kp/model.py` | Контракт входа: результат расчёта калькулятора (`CalcResult`) + коммерческие допущения |
| `kp/compose.py` | Смета поверх результата: скидка, НДС, ₽/м², доставка, проектирование, монтаж, график оплаты |
| `kp/render.py` | Рендер HTML-страницы КП |
| `kp/__main__.py` | CLI |
| `web/` | Интерактивное веб-демо (React + TypeScript) |

**Вход — результат расчёта калькулятора НВФ ИНСИ** ([fasad](https://github.com/kilevoy/fasad)): сводная спецификация и итоги. КП не пересчитывает материалы, а накладывает коммерческие слои — поэтому цифры совпадают с калькулятором 1-в-1.

## Тесты

```bash
pytest tests/ -q
```

Покрывают геометрию, спецификацию, смету (НДС, скидка, ₽/м², график оплаты) и рендер. Запускаются в CI.
